# Book Library Collection

A React app showcasing a collection of books using reusable components, built with Vite.

## Components
- BookCard: Reusable component to display individual book information

## Links
- **GitHub Repository:** [https://github.com/fathimaruhi/book-library.git]
- **Live Website:** [https://dreamy-faun-2d6c7e.netlify.app/]

## Run Locally
1. Clone the repository
git clone [https://github.com/fathimaruhi/book-library.git]
2. Run `npm install`
3. Run `npm run dev`
